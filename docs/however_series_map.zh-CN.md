# however 系列仓库地图

| 仓库 | 角色 | 说明 |
|---|---|---|
| `howeverpipecat` | 语音与多模态智能体框架发行版 | 实时对话 pipeline、服务接入与工程化治理 |
| `howeverautoware` | 自动驾驶 ROS2 集成层 | 车辆栈集成、参数治理、回归流程 |
| `howevercarla` | 仿真与验证底座 | 场景仿真、实验复现、服务健康检查 |
| `howeverapollo` | 自动驾驶平台工程 | 平台化改造与主栈工程实践 |
| `howeverpilot` | 端侧运行时 | 设备接入、运行监控、发布执行 |

## 关系说明

1. `howeverpipecat` 关注 AI Agent 实时交互与工程可维护性。
2. 自动驾驶相关仓库属于并行技术线，文档可互相借鉴，但代码与依赖独立。
3. 统一品牌前缀 `however` 用于发行版识别和仓库归档管理。
